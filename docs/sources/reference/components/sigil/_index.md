# `sigil`

This section contains reference documentation for the `sigil` components.
These components receive and forward generation records for [Grafana AI Observability](https://grafana.com/docs/grafana-cloud/machine-learning/ai-observability/), in the wire format produced by the [Grafana AI Observability SDK](https://github.com/grafana/sigil-sdk).
